# 📧 Email Sender API

A simple **Node.js API** for sending OTP emails with **Express** and **Nodemailer**.

## 🌟 Features
- 📩 Send OTP emails
- ✅ Validates email format before sending
- 🔐 Uses `.env` for secure configuration
- 🚀 Supports **nodemon** for development
